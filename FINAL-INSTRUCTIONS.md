## 🆘 Troubleshooting

### "npm is not recognized"
**Solution**: Install Node.js from https://nodejs.org

### "Cannot find module"
**Solution**:
```powershell
rm -r node_modules
npm install
```

### "Build failed"
**Solution**:
```powershell
npm run build
```
Check for errors, then:
```powershell
npx vercel --prod
```

### "Deployment failed"
**Solution**:
```powershell
npx vercel --prod --force
```

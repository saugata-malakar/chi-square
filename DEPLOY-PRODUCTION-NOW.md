## 🚀 DEPLOY IN 3 STEPS

### Step 1: Navigate to Production App
\`\`\`powershell
cd "C:\Users\trina\Downloads\files (2)\production-app"
\`\`\`

### Step 2: Setup Environment Variables

1. Copy the example file:
\`\`\`powershell
copy .env.example .env.local
\`\`\`

2. Get your FREE API keys (15 minutes):
   - **Supabase**: https://supabase.com/dashboard → New Project
   - **Clerk**: https://dashboard.clerk.com → New Application
   - **Gemini**: https://makersuite.google.com/app/apikey
   - **PostHog**: https://app.posthog.com/signup
   - **Sentry**: https://sentry.io/signup
   - **Upstash**: https://console.upstash.com → New Redis
   - **Pinecone**: https://app.pinecone.io → New Index

3. Edit `.env.local` with your keys

### Step 3: Deploy!
\`\`\`powershell
deploy.bat
\`\`\`

**Done!** Your app is live in 5 minutes.
